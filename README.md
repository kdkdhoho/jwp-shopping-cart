# 비즈니스 요구사항

- [x] 상품 목록 페이지 연동
  - [x] "/" 로 접근하는 경우 상품 목록 페이지를 조회한다.
  - [x] 상품 목록을 클릭하면 상품 목록 페이지를 조회한다.
- [x] 관리자 도구 페이지 연동
  - [x] "/admin" 로 접근하는 경우 관리자 도구 페이지를 조회할 수 있다.
- [x] 상품 관리 CRUD API 작성
  - [x] 상품을 추가할 수 있다.
  - [x] 상품을 수정할 수 있다.
  - [x] 상품을 삭제할 수 있다.
- [x] 상품이 가지는 정보는 다음과 같다.
  - 상품 ID
  - 상품 이름
  - 상품 가격
  - 상품 이미지
